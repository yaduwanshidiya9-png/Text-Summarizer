# Text-Summarizer

A smart text summarizer built using NLP techniques that transforms lengthy content into short, informative summaries while preserving the core meaning.  

## Features

- Converts long paragraphs/articles into short, meaningful summaries.  
- Preserves the core context and important information of the original text.  
- Simple and user-friendly web interface using HTML.  
- Backend implemented in Python with NLP-based summarization logic.  
- Pre-trained model saved and loaded from `saved_summary_model`.


## Tech Stack

- **Language:** Python

- **Frontend:** HTML (`index.html`)

## Installation

1. **Clone the repository**

```bash
git clone https://github.com/yaduwanshidiya9-png/Text-Summarizer.git
cd Text-Summarizer
```

2. **Create and activate virtual environment (optional but recommended)**

```bash
# Create virtual environment
python -m venv venv

# Activate (Windows)
venv\Scripts\activate

# Activate (Linux / macOS)
source venv/bin/activate
```

3. **Install dependencies**

```bash
pip install -r requirements.txt
```

## Usage

### 1. Run the web app

```bash
python app.py
```

- Application local URL (example): `http://127.0.0.1:5000/`  
- Open this URL in your browser.  
- Enter or paste your long text in the input area.  
- Click on the "Summarize" button to generate the summary.  

> Note: Exact route / port may vary depending on `app.py` implementation.

### 2. Use the Notebook

If you want to see how the model / logic works:

```bash
jupyter notebook text_summarizer.ipynb
```

- Explore data preprocessing, feature extraction, and model training steps.  
- You can modify and experiment with different summarization approaches inside the notebook.  

## How It Works (High Level)

- The input text is preprocessed (cleaning, tokenization etc.).  
- Important sentences / parts of the text are identified using NLP-based techniques.  
- A short summary is generated by selecting the most relevant information while maintaining coherence.  
- The trained model is saved under `saved_summary_model/` and used by `app.py` for inference.
